HTML Coding Journal

1. HTML starts and ends with <html> and </html>.
2. CSS is actually part of HTML. It's like HTML's makeup.
3. <h1>, <h2>, <h3>, <h4>... different headings. Smaller number = bigger size.
4. Use <p> and </p> for paragraphs.
5. The style attribute in HTML is used for CSS: <h1 style="color:blue;">words</h1>
6. There are many styles, like changing background color with background-color:powderblue;. Note: styles use ":" and ";" to separate them.
7. Before starting, add <!doctype HTML> and wrap your code in <html> and <body> tags.
8. Use <br> for line breaks.
9. For maintaining spaces and line breaks, use <pre> and </pre>.
10. Attributes like color, size, src, href are important.
11. src is used for images, href for links: <a href="link">something</a>.
12. HTML elements can be styled similarly.
13. Sometimes there may be extra spaces; not sure why. Check your code.
14. For today, write about images and font sizes.
15. Use <img src="image.format" width="xxx" height="yyy"> to set images. src is the source.
16. Font sizes and styles:
    <b> - Bold text
    <strong> - Important text
    <i> - Italic text
    <em> - Emphasized text
    <mark> - Marked text
    <small> - Smaller text
    <del> - Deleted text
    <ins> - Inserted text
    <sub> - Subscript text
    <sup> - Superscript text
17. Styling borders with colors: <h1 style="border:2px solid Tomato;">hello apurbo</h1>. 2px is the size.
18. Set colors using color:tomato;.
